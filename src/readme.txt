config 	- compiler and library configuration files
dll	- for windows: the dynamic link library build project
frb	- fuzzy rule bases and systems for inference: this has been abandoned
gis	- wrappers around GDAL to comply with prim, etc.: these have been pushed out in other projects
io	- streams, binary streams, matlab streams, ...
lib	- the compiled output
num	- numeric primitives and functions
prim	- geometric primitives
spat	- spatial subdivisions, quadtrees, octrees, meshes in 2D and 3D, triangulators, ...
util	- Python wrapper, general utility, debug facilities, ...
stde	- extensions to the STL 
test	- the test suite
