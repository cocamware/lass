config 	- compiler and library configuration files
ext	- directory containing external code and headers, like boost, xerces, gdal, ...
frb	- fuzzy rule bases and systems for inference
gis	- wrappers around GDAL to comply with prim, etc.
io	- streams, binary streams, vrmlstreams, ...
	io\db	- general database system
lib	- the compiled output
num	- numeric primitives and functions
prim	- geometric primitives and basic functionality
spat	- spatial subdivisions, quadtrees, octrees, meshes in 2D and 3D, triangulators, ...
util	- general utility, debug facilities, ...
